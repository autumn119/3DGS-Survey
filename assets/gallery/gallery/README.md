# Gallery assets

Place your Drum Tower dataset renderings here:

- `zeli_rendering.png` — 3DGS rendering of Zeli Drum Tower
- `chao_li_rendering.png` — 3DGS rendering of Chao Li Drum Tower
- `comparison.png` — Ground truth vs. rendering comparison

Replace the `img-placeholder` blocks in `index.html` with actual `<img>` tags pointing to these files.
